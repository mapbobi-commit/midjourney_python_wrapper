# MidJourney API Python wrapper

Python client for Unofficial MidJourney Wrapper

## Requirements:

- python
- pip

## How to run

### Virtual environment

- `$ python -m venv .venv`
- `$ source .venv/bin/activate`
- `$ python -m pip install -r requirements.txt`

### Run

Put your own codes into config.yaml and then run:

- `py main.py`
